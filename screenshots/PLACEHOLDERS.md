#  Application Screenshots

Add your actual screenshots here by:

1. Taking screenshots of your running application
2. Saving them as PNG or JPG files in this folder
3. Updating the README.md to reference the actual files

Recommended screenshots:
- login.png - Login/registration screen
- main-chat.png - Main chat interface
- private-chat.png - Private messaging
- file-sharing.png - File upload and sharing
- mobile-view.png - Mobile responsive design
- notifications.png - Notifications in action

To take screenshots:
1. Run your application (npm start in both client and server)
2. Open http://localhost:3000
3. Take screenshots of different features
4. Save them in this folder with descriptive names
